Connect Claude
To connect Claude.ai to your game library via MCP:
1. In Claude.ai → Settings → Integrations → Add MCP Server
2. Server URL: https://your-domain/mcp/[MCP_TOKEN]
3. No additional auth header needed — the token is in the URL
4. Once connected, Claude can read your library and profile
5. Ask Claude to evaluate games, suggest moves, or update your profile
6. All suggestions appear in the Pending tab for your approval
MCP_TOKEN is set in your .env file on the server.
